# Mock Cross-border Gateway

A lightweight simulated cross-border payment gateway built for processing international transactions.

## Tech Stack
* **Language:** Java 17
* **Build Tool:** Apache Maven
* **Network & IO:** Java native HttpClient
* **Data Parsing:** Google Gson
* **Deployment:** AWS EC2 (Amazon Linux 2023)

## Core Features
* Simulating POST requests for order creation.
* Parsing third-party JSON webhooks.
* Automated build process with Maven.
